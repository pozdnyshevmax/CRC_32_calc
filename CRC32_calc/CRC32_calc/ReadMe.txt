========================================================================
    КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Обзор проекта CRC32_calc
========================================================================
Это приложение реализует CRC32 калькулятор.

Приложение выбирает файлы по введенной маске, проводит вычисления и записывает 
результат в файл с префиксом "result_".

параметры передаются таким образом - 
crc_32 MASK [someBeginTag someEndTag]*

примеры - 
crc_32 *.txt lol /lol
crc_32 *.txt pss pss2 lol /lol
